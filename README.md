# SwiftUICentered

## Instalation

To install this package, import `https://github.com/RozhinaNadya/SwiftUICentered.git` in SPM

## Usege example

```swift

struct ContentView: View {
    var body: some View {
        VStack {
            Text("Centered")
            .centerHorizontally()
        }
    }
}
```
